
# IBGE Location 

## :bulb: Projeto

IBGE Location é uma aplicação que busca dados da [API](https://servicodados.ibge.gov.br/api/v1/localidades/estados) pública do IBGE no Brasil e inicia uma busca no Google de acordo com os dados informados.
